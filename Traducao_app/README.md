# Website React+Typescript+Vite+Bootstrap5.0

Site prático para serviços de traduções, auxiliando o cliente a estar recebendo traduções de forma mais rápida e eficiente.

## Funcionalidades

-Cadastro de clientes
-Painel de Dashboard cliente cadastrado
-Envio de arquivos JPG/JPEG/PDF/DOC
*Cada documento enviado fica cadastrado no banco de dados com ligado ao cliente que o enviou
*os arquivos ficam gravados em uma pasta no backend/uploads
-Painel de Dashboard para o admnistrador do sistema
*com acesso aos clientes
*mensagens individual e coletivo para clientes cadastrados
\*visualização de pagamentos online

## Pré-requisitos
